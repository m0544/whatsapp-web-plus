# Quick Replies – User Flow (Phase 2)

## Create
1. User opens **Dashboard → Quick replies** (`/dashboard/quick-replies`).
2. Clicks **Add template** (הוסף תבנית).
3. Enters **Shortcut** (e.g. "On my way") and **Content** (the message text).
4. Clicks **Add** → template appears in the list. Dialog closes.

## Use
- **From Home (when connected):** Click **Send quick reply** → dialog opens with phone field + list of templates → enter number, click a template → message sends.
- **From Send page:** Enter phone → choose a template → click to send.

## Manage
- Same dashboard: list shows all quick replies (shortcut + content preview).
- **Delete:** Click delete on a row → template is removed. No edit in MVP (delete + re-add if needed).

## Principles
- Single place to **manage** templates; multiple **entry points** to **send**.
- Minimal steps: create once, use from home or send page without extra navigation.
